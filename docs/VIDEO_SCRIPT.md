# Silent cursor-only demo script

Recommended duration: 60-90 seconds. No voice needed.

## Screen sequence

1. Open the public ngrok/Vercel URL.
2. Pause on the hero text: `Pay many contributors from one treasury token`.
3. Move cursor to `Treasury setup` and keep `Sender spends: TON` selected.
4. Add or import recipients. Show different receive tokens: USDT, TON, STON.
5. Connect a TON wallet. Do not use demo wallet for the final submission recording unless the real wallet flow is unavailable.
6. Click `Quote all through Omniston`.
7. Pause on the route text so viewers see `TON → Omniston → recipient token`.
8. Click `Pay` for one safe small payout or show wallet review.
9. Return to the page and show the live receipt.
10. Click `Export Excel receipt`.

## Optional text captions

- `One treasury token`
- `Many recipient-preferred assets`
- `Quoted through STON.fi Omniston`
- `Reviewed in TON wallet`
- `Readable payout receipt exported`
